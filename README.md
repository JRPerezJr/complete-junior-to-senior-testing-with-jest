### 🏠 [Homepage](https://github.com/JRPerezJr/complete-junior-to-senior-testing-with-jest)

## 📐 Prerequisites

- node 14.17.0
- npm 6.14.13
  

## 🛠 Install

```sh
yarn
```

## 💻 Usage

```sh
yarn start
yarn async
```

## ⚙️ Dev mode
```sh
yarn dev
```

## Run 🧪tests

```sh
yarn test
```
